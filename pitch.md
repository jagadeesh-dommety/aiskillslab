Embrace AI in Interviews, 

Currently there is a identified gap in market hiring, where the real work being pushed towards the AI assistance and not writing code by hand, analytics with AI where as on other hand the interviews are still using the traditional way without having any tools, memorization. Rather avoid the AI, we want the enterprises to embrace. The org wanted to change the pattern but there is a lack of support of tools, internal hiring acceptance, lacking SOP and proper problem and evaluation. Our product will provide a tools, SOP's, problems bank without disrupting the existing interview process. 

The product doesnt command what to do, rather what can be done

Before discussing the AI role in interview, wanted to put boundries and what AI not doing
1. AI not replacing the human interviewing process, it is assisting both candidate and interviewer
2. No judgement from AI
3. No video and audio recording and analysis
4. No keystroke analysis
5. No PII involved, if so, all data will be deleted in 30 days
6. Not removing whiteboard discussions in with interview

What the AI native labs provide is
1. A repository bank having huge set of repo's and problems - predefined maintained by team and keeping up to date and removing stale issues
2. The platform to solve the problems, for example dev box, IDE, AI tools, logging the user file and ai navigation
3. Cloud tools - like azure, aws and gcp resources
4. Interviewer portal - repo browser, session launche, interviewer can provide the level and tech needed and repo will be selected
5. Telemetry providing to interviwer on how candidate prompts, navigation, whiteboard anlysis


In addition to the traditional Interviewer and candidate, there will be AI assistant for both Interview and Candidate

What is been acheived on this
1. Candidate not stripped of the tools - which is like a day job simulation
2. Interview is no longer the memory and recall based
3. Candidate is no longer the code producer, rather a decision maker
4. As already been provided with tools, there will be less suspicion of cheating in interviews
5. Creating future interviwes
6. Interviwer been provided with more data
